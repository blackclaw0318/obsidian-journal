## 🛠 技术栈

- **前端**: Next.js 14 (App Router) + React 18 + TypeScript
- **样式**: Tailwind CSS + 自定义主题
- **数据库**: SQLite + Prisma 5.22 + WAL 模式
- **测试**: Vitest 2.1 + Playwright 1.49
- **部署**: PM2 + nginx + Cloudflare Tunnel (开发演示)

## 🚀 快速开始

```bash
# 1. 安装依赖 (国内用户需先设置镜像)
npm config set registry https://registry.npmmirror.com/

# 2. 一键启动 (db push + seed + dev)
npm run db:reset
npm run dev

# 3. 访问
open http://localhost:3000

# 4. (可选) 暴露给老板看
npm run dev:tunnel
```

## 📊 数据模型 (14)

```
核心内容:    Post, Chapter, Novel, NovelVolume, Video, VideoSeries
媒体:        MediaItem, MediaUsage
结构化:      Page, Block (13 种 JSONB)
配置:        SiteConfig, Social, User, Session
```

## 🧱 Block 体系 (13)

```
基础:    text, heading, image, video, gallery
排版:    quote, callout, code, divider
列表:    list, table
高级:    custom_html (默认禁), music (advanced), embed
```

## 🎬 一键命令

```bash
npm run dev          # 本地开发
npm run build        # 生产构建

npm run test:e2e     # 跑 E2E (Playwright)
npm run verify       # 全量验证 (typecheck + lint + test + e2e)
npm run dev:tunnel   # Cloudflare Tunnel (老板本地访问)
```
